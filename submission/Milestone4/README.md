\# Milestone 4: Dashboard \& Deployment



\## Objective

To deliver an enterprise-ready AI-powered email classification system with real-time prediction, visualization, and deployment capability.



---



\## Features Implemented

\- Email category prediction (Spam / Not Spam)

\- Urgency detection (High / Medium / Low)

\- Interactive web dashboard

\- Simulated real-time email stream

\- Deployment-ready structure



---



\## System Architecture

1\. Backend ML models for category \& urgency prediction

2\. Streamlit-based interactive dashboard

3\. Real-time email testing via simulated streams

4\. Cloud deployment using Streamlit Cloud



---



\## Dashboard

\- Users can input email text

\- System predicts:

&nbsp; - Email category

&nbsp; - Urgency level

\- Displays sample dataset for reference



---



\## Technologies Used

\- Python

\- Scikit-learn

\- Streamlit

\- Pandas

\- TF-IDF Vectorization
